## [[Estructura Lineal]]
- Secuencias de datos
- Todo elemento tiene un único predecesor (salvo el primero)
- Todo elemento tiene un único sucesor (salvo el último)
## [[Estructura Jerárquica]]

cualquier elemento puede tener
- Varios predecesores
- varios sucesores
